About This Project

I built the SysOps Toolkit as a portfolio project to showcase my hands on IT and cybersecurity skills developed through professional experience and certification training. Each module aligns with CompTIA A+, Network+, and Security+ exam objectives covering hardware troubleshooting, network infrastructure, and security operations. The toolkit is developed using Python, Flask, and open-source libraries with all modules designed to run locally without external API dependencies. This project reflects my ability to apply real-world troubleshooting, networking, and security concepts in a practical environment.
